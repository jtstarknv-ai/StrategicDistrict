4. The Approach

Every icon is built as inline SVG using our existing color palette (teal, gold, green, coral, purple, navy). No external image files needed. They render crisp at any size, load instantly, and can be themed. The glassmorphism effect comes from layered opacity, soft gradients, and subtle strokes — not heavy blur effects that would slow things down.

Zero Dependencies
Pure SVG. No image files, no CDN, no loading delays. Deploys with FTP like everything else.
On-Brand Colors
Uses the exact CSS variables already in the design system. Teal for builder, green for dashboard, coral for school.
Subtle, Not Loud
Translucent fills, thin strokes, soft accents. Professional enough for superintendent presentations. Clean enough for daily use.
Where They Go
Header nav links, sidebar step list, empty states, export covers, and the landing/login page. Targeted placement, not icon soup.
